# Resort Luxury
Criação de Website inspirado em resorts.

## Sobre o Projeto
Esse projeto tem finalidades acadêmicas e didáticas, trata-se da criação de um website que mostra destinos para possiveis viagens.

## Processo de Criação

- HTML5 Semântico
- CSS3
- JavaScript
- Flex Box
- CCS Grid
